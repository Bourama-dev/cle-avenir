# Variables Google Tag Manager

## Variables de données

### User Data
- user_id: ID utilisateur (si connecté)
- user_type: Type d'utilisateur (élève, étudiant, etc.)
- user_region: Région de l'utilisateur

### Test Data
- test_duration: Durée du test en secondes
- test_answers: Nombre de réponses
- test_completion_rate: % de complétude

### Results Data
- top_career: Métier principal recommandé
- top_career_percentage: % de correspondance
- top_5_careers: Liste des 5 métiers
- user_vector: Vecteur utilisateur

## Configuration dans GTM

1. Aller dans "Variables"
2. Créer une variable "Data Layer Variable"
3. Entrer le nom de la variable
4. Utiliser dans les événements
