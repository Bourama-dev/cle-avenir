# Plan d'Apprentissage Intelligent - CléAvenir

## 1. Système de Logging

### Données à collecter

#### A. Données utilisateur
- Email / ID utilisateur
- Profil (âge, niveau d'études, région)
- Réponses au test (question + réponse)
- Temps passé par question
- Modifications de réponses

#### B. Données de résultats
- Vecteur utilisateur final
- Top 5 métiers recommandés
- Pourcentages de matching
- Raisons du matching
- Confiance du test

#### C. Données comportementales
- Métiers cliqués
- Formations consultées
- Temps passé sur les résultats
- Conversions (inscription, etc.)
- Feedback utilisateur

### Format de logging
