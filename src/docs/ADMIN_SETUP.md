# Configuration Administrateur

## Email Admin

L'email administrateur est : **contact@cleavenir.com**

Toute connexion avec cet email donne automatiquement les droits admin.

## Accès Admin

1. Se connecter avec contact@cleavenir.com
2. Être automatiquement redirigé vers /admin/dashboard
3. Accès complet à :
   - Tableau de bord
   - Gestion des utilisateurs
   - Statistiques détaillées
   - Paramètres système
   - Logs techniques
   - Données ML

## Permissions Admin

✅ Lecture complète des données
✅ Écriture/modification des contenus
✅ Suppression de données
✅ Gestion des utilisateurs
✅ Accès aux statistiques
✅ Configuration système
✅ Export de données
✅ Accès aux logs

## Permissions Utilisateur Standard

✅ Voir son profil
✅ Faire le test
✅ Voir ses résultats
✅ Consulter les formations
✅ Consulter les métiers
❌ Pas d'accès admin
❌ Pas de gestion d'utilisateurs
❌ Pas de statistiques globales

## Sécurité

- Le rôle admin est basé sur l'email
- Aucune dépendance à la base de données
- Vérification à chaque connexion
- Badge visible dans l'interface
